# Nuwa UI Render

A lightweight, framework-agnostic web component designed to render content blocks created with the Nuwa Software visual editor. This component is optimized for performance and can be deployed in any environment that supports standard web components.

## Features

*   **Lightweight:** Minimal footprint for fast loading and rendering.
*   **Framework-Agnostic:** Use it with any framework (or no framework at all).
*   **Deploy Anywhere:** Easily deploy to any hosting service that serves static content.
*   **Nuwa Compatible:** Specifically designed to interpret and render content from the Nuwa Software visual editor.

## Getting Started

1.  **Install the component:**
    ```bash
    npm install
    ```
2.  **Run the development server:**
    ```bash
    npm run dev
    ```
3.  **Build for production:**
    ```bash
    npm run build
    ```
